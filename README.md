# Homebrew Tap for Antify

Per-app transparent proxy for macOS. Route specific apps through SOCKS5/HTTP proxies without system-wide global proxy.

## Install

```bash
brew tap free-lester/antify
brew install --cask antify
```

## Update

Antify supports in-app auto-update via Sparkle. If you prefer updating through Homebrew, use:

```bash
brew upgrade --greedy --cask antify
```

## Uninstall

```bash
brew uninstall --cask antify
brew untap free-lester/antify
```

## About

- Homepage: [antifyapp.com](https://antifyapp.com)
- Requires macOS 14.0 (Sonoma) or later
- After installation, approve the System Extension in **System Settings > Privacy & Security**
